## Тестовая задача для Digital–студии: Sepium

### Общие требования:

- **Объем**: Необходимо сверстать листинг с мини-карточками (заверстать нужно
только 1 мини-карточку и продублировать ее).
- **Адаптив**: Верстка должна быть кроссбраузерной (одинаково хорошо
отображаться во всех браузерах) и адаптивной (десктоп и мобила).
- **Стек**: При работе необходимо использовать чистый HTML, CSS, JS/jQuery,
Fancybox, любую библиотеку для создания слайдера.
- **Методология**: Необходимо придерживаться принципов БЭМ и PixelPerfect.

### Запрещается:
- Использование html-шаблонизаторов и css-препроцессоров.
- Использование фреймворков (React, Vue.js и проч.).
- Самописный (кастомный) код для слайдера.
